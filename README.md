# Steam Backlog Triage

Консольное приложение на C# (.NET 9), которое анализирует вашу библиотеку Steam, подтягивает оценки с Metacritic и время прохождения из HowLongToBeat, а затем:

- строит **список игр до N часов** (по умолчанию до 20),
- исключает мультиплеерные/бесконечные игры,
- убирает проекты, которые вы уже почти прошли,
- формирует **красивый HTML-отчет** с сортировкой по колонкам и ссылками на Steam и HowLongToBeat.

Подходит, если у вас сотни игр в библиотеке и вы хотите быстро выбрать, **во что поиграть дальше**.

## Основные возможности

- Получение списка всех игр с вашего аккаунта Steam (через Steam Web API).
- Загрузка оценок Metacritic для каждой игры.
- Загрузка времени прохождения из HowLongToBeat (через `Codepotatoes.Scraper.HowLongToBeat`).
- Кэширование результатов в локальный файл `cache.json`.
  
- Фильтрация:
  - только игры с известным временем прохождения,
  - только игры до 20 часов (по умолчанию),
  - исключение чистого мультиплеера и бесконечных игр,
  - исключение уже фактически пройденных игр (если наиграно > 80% от HLTB).
    
- Генерация HTML-страницы со:
  - списком игр,
  - обложками из Steam,
  - оценками Metacritic с цветами (зелёный/жёлтый/красный),
  - временем HLTB и наигранными часами,
  - кликабельными ссылками на Steam и HowLongToBeat,
  - интерактивной сортировкой по заголовкам таблицы.

## Технологии и архитектура

- **.NET 9.0**, C#, консольное приложение.
- Пакеты:
  - `Codepotatoes.Scraper.HowLongToBeat` — работа с HowLongToBeat.
- Архитектура:
  - `Program` — только UI (меню, ввод, прогресс-бар).
  - `SteamBacklogService` — сбор и объединение данных (Steam + HLTB).
  - `CacheService` — чтение/запись `cache.json`.
  - `GameFilter` — все бизнес-правила по выбору игр.
  - `HtmlReportGenerator` — генерация HTML-отчета.

## Подготовка и требования

### 1. Необходимое ПО

- [.NET SDK 9.0](https://dotnet.microsoft.com/) (или выше).
- Windows x64 (по умолчанию проект собран под `win-x64`).

### 2. Получение Steam API Key (один раз)

Steam API Key нужен, чтобы получить список игр с вашего аккаунта.

1. Откройте сайт:  
   `https://steamcommunity.com/dev/apikey`
2. В поле **Domain Name** можно ввести что угодно, например:  
   `localhost`
3. Нажмите **Register** и скопируйте выданный 32-значный ключ.
4. При первом запуске программа сама предложит либо:
   - открыть эту страницу в браузере,
   - либо сразу вставить ключ в консоль.
5. Ключ один раз сохранится в `config.json`.

> **Опционально:** Не забудьте после работы программы отозвать SteamAPI токен.

### 3. Настройки приватности в Steam

Чтобы программа видела ваши игры, в настройках профиля Steam нужно:

- Профиль: **Публичный**
- Детали игры: **Публичный**
